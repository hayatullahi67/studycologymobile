# AceCBT Mobile

This is a native mobile application built with **Expo** and **React Native**.

## Development

**Prerequisites:** Node.js, Expo Go app (on your phone) or an emulator.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```

3. Open the app:
   - **Scan the QR code** with Expo Go (Android) or your Camera app (iOS).
   - Press **a** for Android emulator.
   - Press **i** for iOS simulator.

## Features
- Practice CBT sessions offline.
- Study notes and past questions.
- Performance history tracking.
